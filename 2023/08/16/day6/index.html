哈希表理论基础

常见的哈希表使用的数据结构：

  • 数组
  • set
  • map

分析待补充

今日题目

  • 242.有效的字母异位词

分析：

题目给定字符串内容全为小写字母，可直接使用数组实现哈希表

  • 349. 两个数组的交集

分析：

数组中元素唯一，同时不要求输出结果的顺序，故考虑使用set结构实现

  • 202. 快乐数

分析：

得到的平方和无序，且快乐数的平方和不重复（若重复则进入了无限循环），故可使用set结构

  • 1. 两数之和

分析：

本题需同时判断两数和是否为目标值，并返回数组下标，可使用map结构
